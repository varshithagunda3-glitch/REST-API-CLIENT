*Task Description

This task involves creating a Java application that consumes a public REST API and displays the retrieved data in a structured and readable format. A REST API (Representational State Transfer Application Programming Interface) allows different software applications to communicate with each other over the internet and exchange data.
In this project, the Java program connects to a public weather API to fetch real-time weather information. The API returns data in JSON (JavaScript Object Notation) format, which is a commonly used format for exchanging data between applications.
The program sends an HTTP request to the weather API server and receives weather data such as temperature, wind speed, wind direction, and time. After receiving the response, the program processes the JSON data and extracts the important values required for display.Finally, the application prints the weather information in a neat and structured report format in the console, making it easy for users to understand the weather details.

*How the Task is Performed

The program defines the API URL that contains the location details.

It creates a connection to the API server using Java networking classes.

The application sends an HTTP request to fetch weather data.

The server returns the response in JSON format.

The program extracts the required data from the JSON response.

The information is displayed in a clear and formatted weather report.

*Tools and Technologies Used

Java Programming Language

Java Networking Classes (URL, URLConnection, BufferedReader, InputStreamReader)

JSON Data Format for API response
